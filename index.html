PROJECTS

  • Incremental FlashFill [In Progress]
    • FlashFill is a Microsoft Excel 2013 feature that fills out columns of the spreadsheet automatically when a pattern is visible based on a few examples provided by the user. The work is explained in Automating String Processing in Spreadsheets using Input-Output Examples, POPL 2011, Sumit Gulwani. We are working on incrementalizing the intersection of Directed Acyclic Graphs, the data structure used to represent string expressions.
  • Predicting and Mapping the Northern Lights
    • Northern Lights (also known as Aurora Borealis) are natural electromagnetic phenomena characterized by the appearance of colored lights in the night sky, usually near the higher latitudes. Using a historical dataset obtained from a network of magnetometers spread across the western hemisphere between the years 1997 and 2015, a supervised machine learning model is trained to predict the intensity of auroral activity on a given day. [technical report]
